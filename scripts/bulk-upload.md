# Bulk Upload Books Guide

This guide will help you upload 30 books to your BookLoom application.

## Option 1: Manual Upload via Admin Interface

1. **Login as Admin**
   - Go to `/admin/books`
   - Make sure you're logged in as an ADMIN user

2. **Upload Each Book**
   - Fill in the form:
     - Title (required)
     - Author (optional)
     - Description (optional)
     - Publication Year (optional)
     - **Category** (select from dropdown)
     - License Type (required)
     - Cover Image (optional - recommended for better presentation)
     - PDF File (required)
   - Click "Upload Book"
   - Wait for upload and analysis to complete

3. **Repeat for all 30 books**

## Tips for Quick Upload

- Prepare all book information (titles, authors, categories) in advance
- Have cover images ready (PNG, JPG, WEBP, max 5MB each)
- Ensure PDFs are under 10MB
- Use appropriate categories for better organization
- Set books to "Public" so they're visible to everyone

## Recommended Categories

Choose appropriate categories for each book:
- Fiction
- Non-Fiction
- Science Fiction
- Horror
- Fantasy
- Mystery
- Adventure
- Classic
- Romance
- Dystopian
